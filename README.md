# 🚗 RC Camera Car (Wireless Surveillance Robot)

## 📌 Overview
The RC Camera Car is a **wireless robotic vehicle** designed for **real-time video monitoring and remote navigation**. The system integrates **embedded motor control** with **live camera streaming**, allowing users to control the vehicle while observing surroundings remotely.

This project demonstrates practical applications in **robotics, surveillance, exploration, and IoT-based control systems**.

---

## ❗ Problem Statement
In many situations such as surveillance, hazardous environment monitoring, and remote inspection, direct human presence is unsafe or impractical. Traditional systems lack **mobility, real-time feedback, and cost-effectiveness**.

This project addresses the need for a **low-cost, wireless, and real-time robotic monitoring system**.

## 🧠 System Architecture

### 🔧 Main Components
- **Microcontroller (ESP32 / Arduino)** – Core control unit  
- **Motor Driver (L298N / L293D)** – Controls DC motors  
- **DC Motors + Wheels** – Vehicle movement  
- **Camera Module (ESP32-CAM / USB Camera)** – Video streaming  
- **Chassis** – Mechanical structure  
- **Power Supply (Battery Pack)** – System power  
- **Wireless Communication (Wi-Fi / Bluetooth)** – Remote control  

---

## ⚙️ Working Principle

1. The user sends control commands via **mobile app or web interface**  
2. Commands are transmitted wirelessly using **Wi-Fi or Bluetooth**  
3. The **microcontroller processes the input signals**  
4. Motor driver controls the **movement of wheels**  
5. The camera module captures **live video feed**  
6. Video is streamed back to the user in real time  
7. User navigates the vehicle based on live visuals  

---

## 🔄 Workflow

User Input → Wireless Communication → Microcontroller Processing → Motor Driver Control → Vehicle Movement → Camera Capture → Live Streaming → User Monitoring
